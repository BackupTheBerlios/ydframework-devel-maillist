Hi Pieter,

In order to work as intended, the YDWeblogAdminRequest::_getCookiePath
function has to be flipped as follows:

replace line 347:
$cookiePath = str_replace( '\\', '/', rtrim( $cookiePath, '/' ) );

with this new line:
$cookiePath = rtrim( str_replace( '\\', '/', $cookiePath ), '/' );
